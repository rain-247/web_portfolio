Skyline Strike    縱向飛行射擊遊戲

Skyline Strike 是以 Unity 製作的縱向飛行射擊遊戲。


玩家操作戰機穿越空中敵陣，閃避攻擊、擊落敵人並挑戰高分，重現經典街機風格射擊遊戲的爽快節奏。

本作品為 Unity 初學練習項目，重點在於整合基本角色控制、碰撞偵測、敵人生成與 UI 顯示功能。


使用技術：
- Unity：場景搭建、物件碰撞、UI 系統管理
- C#：角色移動控制、敵人與子彈生成、分數與血量計算
- 簡易像素素材與 GUI 組件整合

遊戲設計概念    基礎實作與視覺練習

本作著重於建立 Unity 裡的 2D 縱向捲軸邏輯，並練習基礎技能整合：飛機控制、子彈發射、敵人隨機出現與碰撞處理。

UI 設計包含分數、血量與結算畫面，整體視覺風格簡潔清晰，強調易於辨識與直覺操作。
飛機與敵機皆使用自繪像素風格，營造輕鬆街機風體驗。
